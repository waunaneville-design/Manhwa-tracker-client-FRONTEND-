# MangaTracker

MangaTracker is a Flask-based web application that helps you track manga reading progress, manage collections, and provide a simple interface for organizing your library.  
It uses Python and Flask as the backend, with npm scripts to simplify running and testing commands.

---

## 🛠 Tech Stack

- **Python 3.14** – Core programming language
- **Flask** – Web framework for building the app
- **Pytest** – Testing framework
- **npm scripts** – For running Python commands via `npm run`
- **Git & GitHub** – Version control and collaboration
